# Automated-Drive-Uploading

# 📂 Smart Document Upload Tracker

Never miss a file upload again — get instant notifications
every time someone uploads to your Google Drive.

## ❌ Problem
- Team shares a Google Drive folder for documents
- No one knows when new files are uploaded
- Have to manually keep checking the folder
- No record of what was uploaded and when

## ✅ My Solution — How It Works
Step 1: Someone uploads any file to Google Drive folder
Step 2: System detects the new upload instantly
Step 3: File name, uploader, and time saves to Google Sheets
Step 4: Email notification sends automatically to the team

## 🔧 Tools Used
- n8n — main automation tool
- Google Drive — where files are uploaded
- Google Sheets — auto-log of all uploads
- Gmail — instant email notification

## 📊 Before vs After

| Before | | After |
|--|--------|-------|
| Knowing about new uploads: Manual checking | | Instant notification |
| Record of uploads: None | | Auto-logged in Sheets |
| Time wasted checking: 30 mins/day | | Zero |
